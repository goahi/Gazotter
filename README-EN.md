# Gazotter
[日本語](https://github.com/saladbowl77/Gazotter)  
Gazotter generate a photo include writing.  
![demo](docs/demo-en.png)


Users can customize these elements:  

- Backgroud color
- Text alignment
- Font family
- Font color
- Font size

- HTML CSS  
- JavaScript  
- [html2canvas](https://html2canvas.hertzen.com)
